    Class ObservablePoint

    可观察的二维坐标点类，当坐标值发生变化时会自动触发 markDirty 回调

    Remarks

    该类主要用于监听和追踪坐标点的变化，常用于需要实时更新变换的场景。大部分情况下，使用 Point 类即可满足需求。
