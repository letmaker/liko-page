    Class Point

    二维坐标点类 提供完整的二维平面坐标点操作功能，包括位置设置、距离计算、角度计算等

    Example

    // 创建坐标点
    const point = new Point(10, 20);

    // 计算两点距离
    const distance = point.distance({ x: 0, y: 0 });

    // 链式操作
    point.set(5, 5).add(10, 10).normalize();

    // 使用临时对象减少内存分配
    const temp = Point.TEMP.set(x, y);
